# Hoverbot2

ROS2 Humble autonomous hoverboard robot

## Hardware
- Raspberry Pi 4 (will upgrade to Pi5)
- RPLidar A1
- BNO055 IMU
- RealSense Camera
- ELP 2MP Camera
- Hoverboard motors + controller

## Workspace Structure
- Dev machine: Ubuntu x86_64 - development & visualization
- Robot: Raspberry Pi 4 - hardware control
